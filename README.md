# IzziHub

Landing page institucional da IzziHub reposicionada para venda de:

- sistemas web sob medida
- landing pages para Google Ads
- aplicativos iOS e Android
- integracoes e automacoes

## Estrutura

- `index.php`: landing principal
- `style.css`: estilos e responsividade
- `script.js`: menu mobile e reveal on scroll
- `robots.txt`: diretrizes de indexacao
- `sitemap.xml`: sitemap basico
- `imagens/`: logo, favicon e assets publicos

## Observacoes

- O formulario foi removido para reduzir spam e concentrar o contato no WhatsApp.
- Arquivos locais de backup, deploy e anotacoes operacionais ficam fora do Git via `.gitignore`.
